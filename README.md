# 🛒 API CRUD - Loja de Eletrônicos

Este é um projeto de estudo de uma **API CRUD** desenvolvida em Python. O objetivo é gerenciar um catálogo de produtos eletrônicos de forma simples e didática.

## 🚀 Tecnologias Utilizadas
* **Python 3.12+** (Rodando no Fedora Linux)
* **FastAPI**: Framework web para criação de APIs.
* **Uvicorn**: Servidor para rodar a aplicação.
* **VS Code**: Editor de código.

## 📦 Funcionalidades (CRUD)
A API permite realizar as quatro operações básicas:
1. **Create (Post)**: Adicionar novos produtos (ID, Nome, Preço).
2. **Read (Get)**: Listar todos os produtos ou ver um específico.
3. **Update (Put)**: Atualizar o preço de um produto existente pelo ID.
4. **Delete (Delete)**: Remover um produto do catálogo.
